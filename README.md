## About this repository

This repository is dedicated to:

- bug reports
- feature requests
- tracking issue status and discussions

No app source code is published here.

## Report a bug

Before opening a bug report:

1. Check if a similar issue already exists.
2. Include your app version and device model.
3. Provide clear steps to reproduce.
4. Add screenshots, screen recordings, or logs if possible.

Issues page:
https://github.com/webybox/unisearch/issues

## Request a feature

Please include:

1. The problem you want to solve.
2. Your proposed solution.
3. The expected user impact.
4. Optional mockups/examples.

## Guidelines

- One issue = one bug or one feature request.
- Keep issue titles short and specific.
- You can write in English or Polish.
- Be respectful and constructive.

## Security

Please do not post sensitive data or security vulnerabilities publicly in Issues.
Contact the maintainer privately for responsible disclosure.
